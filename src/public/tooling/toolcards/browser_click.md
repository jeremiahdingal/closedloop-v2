# Tool: browser_click

Use for:
- Clicking an element on the current page
- Interacting with buttons, links, or checkboxes

Arguments:
- selector: string (CSS selector)
- modifiers: string[] (optional: 'Alt', 'Control', 'Meta', 'Shift')

Returns:
- Success message

Example:
{"tool_name":"browser_click","arguments":{"selector":"button#submit"}}
