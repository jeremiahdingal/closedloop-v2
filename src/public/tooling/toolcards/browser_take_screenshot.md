# Tool: browser_take_screenshot

Use for:
- Taking a screenshot of the current page for visual verification
- Capturing the full scrollable page or a specific view

Arguments:
- filename: string (e.g. 'screenshot.png')
- fullPage: boolean (optional, default false)

Returns:
- Success message with path

Example:
{"tool_name":"browser_take_screenshot","arguments":{"filename":"homepage.png","fullPage":true}}
