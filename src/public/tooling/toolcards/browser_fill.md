# Tool: browser_fill

Use for:
- Filling an input field with text instantly (no character-by-character typing)
- Clearing and setting a new value for a field

Arguments:
- selector: string (CSS selector)
- text: string

Returns:
- Success message

Example:
{"tool_name":"browser_fill","arguments":{"selector":"input[name='email']","text":"test@example.com"}}
